Install AndyMOTE Qt Server

Install Library liblirc as described below:

$ sudo apt-get update
$ sudo apt-get install liblirc-dev

Next, install git, goto your home directory & clone the andymoteserver repository

$ sudo apt -y install git
$ cd ~
$ git clone https://github.com/orangereaper/qtAndymoteServer

then compile the source

$ cd qtAndymoteServer
$ qmake -o Makefile qtAndymoteServer.pro
$ make

Move the resulting file to a convenient location; eg:

$ sudo mkdir -p /opt/andymoteserver
$ sudo mv qtAndymoteServer /opt/andymoteserver/andymote

Tidy up

$ cd ~
$ rm -Rf qtAndymoteServer
