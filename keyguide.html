Key Guide

Choice of Key Name when you create your LIRC Config files is important for the best AndyMOTE experience; it is recommended that you use names selected from the table below (Key Names) whenever sensible to do so.
AndyMOTE uses the Key Name to determine which Icon to assign to the Remote Key and the default position for the Key on the Remote (See Table: Default Layout). If AndyMOTE does not recognise the Key Name (ie it does not exist in the Key Name Table) then the Key will be placed at the bottom of the Remote and no Icon will be assigned.
Once a Remote has been created, any Key can be moved into any position you want on the Remote; you are also able to assign any Icon you like to the button (including your own icons). See the AndyMOTE User Guide for details on how to do this.

Important

For AndyMOTE to operate correctly the following LIRC nomenclature assigments are required where these keys exist on the Remote Control:

Device Key LIRC Nomenclature
Volume Up KEY_VOLUMEUP
Volume Down KEY_VOLUMEDOWN
Mute KEY_MUTE
Power Key KEY_POWER
