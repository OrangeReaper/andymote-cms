Edit Device

To edit a Device press the button to the right of the required item on the Navigation Menu; this will open a screen similar to that shown in the figure.

Here you can:

  • Change the name of the Device,
  • Change or remove the icon that should be displayed in the Navigation Menu,
  • Set the On/Off state of the Device.
  • Change the Default Transmit Duration of Keys for this remote.

Notes:

  1. Changing the Device On/Off state does not control any devices; this command merely changes the internally stored state for the Device (see Device On/Off States)
  2. Changing the Default Transmit Duration will modify this setting for every key on the remote (even those you may have changed manually)

Once you have completed your modifications, press the Done button.

Pushing the Back button at any time will cancel the operation.

Edit Kodi Device

If the device you have chosen to edit is a Kodi Device then you will see an additional option to change the IP address of your kodi device and the option to Change the Default Transmit Period is not there (not required).

Push the Test Button to check the connection between the App and your Kodi Device. A green banner will appear on the screen if successful. If a red banner appears then no connection could be made; in this case check that:

  • the Kodi Device is up, running and configured correctly,
  • the IP address is set correctly,
  • the wifi on your Mobile/Tablet is turned on, and
  • you are connected to the same local network as the Kodi Device.
